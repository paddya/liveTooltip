jQuery.liveTooltip()

This plugin provides an easy way to add tooltips to elements.

Examples

This first example is a quite simple usecase: it renders the title-attribute of the links in a beautiful way (in contrast to the default browser-tooltip).

Demo

  • Google
  • Facebook
  • Github

Code

$(document).ready(function() {
	$('ul li a').liveTooltip({
		contentHandler: function(tooltip) {
			tooltip.text($(this).data('_title')); // title is stored in element.data to prevent 
the browser-tooltip while hovering the element
		}
	});
});
