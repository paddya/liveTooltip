jQuery.liveTooltip()

This plugin provides an easy way to add tooltips to all elements which match the given selector.

Usage

$('selector').liveTooltip(options);

Options

Parameter Type Default Description
contentHandler callback $.noop This function is supposed to fill the tooltip with content. The tooltip is passed as jQuery-object, this references the element which triggered the tooltip.
fadeSpeed string | integer false fade speed (directly passed to .fadeIn() and .fadeOut())
startEvents string mouseenter One or more events which trigger the tooltip.
stopEvents string mouseleave One or more events which destroy the tooltip.
absolutePositioning boolean false If set to true the tooltip is positioned absolute, not relative to the cursor.
position object {x: 20, y:20} The initial position of the tooltip (depending on your absolutePositiong-setting)
trace boolean true If set to true, the tooltip position is updated when the user moves the cursor.
delay number (in ms) 0 Delay
className string '' adds one or more custom CSS-classes to the tooltip.

Examples

This first example is a quite simple usecase: it renders the title-attribute of the links in a beautiful way (in contrast to the default browser-tooltip).

Code

$(document).ready(function() {
    $('#linklist li a').liveTooltip({
        contentHandler: function(tooltip) {
            /* this plugin automatically moves the title-attribute of
             * the affected elements to .data('_title') to prevent the
             * browser-tooltip while hovering the link
             */
            tooltip.text($(this).data('_title'));
        }
    });
});
